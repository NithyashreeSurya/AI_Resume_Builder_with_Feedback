# AI Resume Builder with Feedback

## 📌 Project Overview

The AI Resume Builder with Feedback is a web-based application designed to help users create professional resumes and improve their quality using intelligent feedback. The system not only allows users to build resumes but also analyzes uploaded resumes and provides suggestions to enhance content, structure, and keyword optimization.

## 🚀 Features

* Resume creation with user input
* Resume upload and analysis
* AI-based feedback and suggestions
* Resume scoring system (out of 100)
* Keyword and action verb recommendations
* ATS-friendly resume improvement

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **AI Integration:** API-based analysis + rule-based logic
* **Tools:** VS Code, GitHub

## ⚙️ How It Works

1. User enters details or uploads a resume
2. The system processes the content
3. AI analyzes grammar, keywords, and structure
4. Feedback and suggestions are generated
5. A resume score is provided

## 🎯 Purpose

The main goal of this project is to help students and job seekers create better resumes that are optimized for Applicant Tracking Systems (ATS) and improve their chances of getting shortlisted.

## 🚧 Limitations

* No database integration
* Basic rule-based scoring in some areas
* Limited customization options

## 🔮 Future Improvements

* Add database for storing resumes
* Improve AI model for better analysis
* Enhance UI/UX design
* Add multiple resume templates
* Deploy on cloud for scalability
